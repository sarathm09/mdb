Markdown Browser

Browse, edit, and present markdown files from your terminal

$ npm install -g @sarathm09/mdb click to copy

Features

Everything you need to work with markdown, right in your browser.

📁

File Explorer

Browse your project's markdown files with a sidebar tree view. Navigate directories and open files instantly.

✎

Markdown Editor

Full-featured editor powered by CodeMirror 6 with syntax highlighting, live preview, and markdown-aware keybindings.

🎬

Presentation Mode

Turn any markdown file into a slideshow with Reveal.js. Split slides with horizontal rules and present directly.

📥

Export

Export your documents as Markdown, HTML, or PDF. Share your work in the format that fits your workflow.

🎨

5 Themes

Choose from five carefully crafted themes including One Dark, Solarized, Dracula, Nord, and GitHub Light.

🔍

Command Palette

Quick access to all commands with a fuzzy-search palette. Find files, switch themes, and toggle modes effortlessly.

⌨

Keyboard Shortcuts

Comprehensive keyboard shortcuts for power users. Navigate, edit, and present without touching the mouse.

📈

Mermaid Diagrams

Render Mermaid diagrams inline. Flowcharts, sequence diagrams, Gantt charts, and more rendered automatically.

Installation

Install with your preferred package manager.

npm

npm install -g @sarathm09/mdb

pnpm

pnpm add -g @sarathm09/mdb

yarn

yarn global add @sarathm09/mdb

bun

bun add -g @sarathm09/mdb

Keyboard Shortcuts

Navigate and control everything from the keyboard.

Shortcut Action
Ctrl + P Open command palette
Ctrl + S Save file
Ctrl + E Toggle editor / preview
Ctrl + Shift + P Enter presentation mode
Ctrl + B Toggle sidebar
Ctrl + Shift + E Open export menu
Escape Close palette / exit presentation
